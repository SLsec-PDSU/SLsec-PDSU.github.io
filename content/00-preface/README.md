# 写在前面

这本手册不是教科书。

它是 SLsec 实验室的学长们把自己踩过的坑、走过的弯路整理出来的东西。你不需要有任何基础，但你需要有耐心——CTF 没有捷径，只有一道题一道题地啃。

本章有三篇文章，建议在开始学任何技术内容之前先读完。

**关于实验室和手册**：你在哪里，这本手册能带你去哪里，以及配套的练习资源在哪里找。

**学习方法**：比任何技术知识都更重要的事。遇到问题怎么办、怎么用好 AI、怎么提问——这些习惯从一开始就要养好。

**免责声明**：认真的。

:::
